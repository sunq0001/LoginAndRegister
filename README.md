## 登陆注册页面 ##

### 基本功能 ###

- 在注册和登陆页面，提交表单之前先判断用户名和密码的正确性
- 用户名是手机或者邮箱格式
- 注册以后，系统发一封验证邮件到用户的邮箱中
- 用户点击邮箱中链接完成验证并且以登陆状态跳转到主页
- 主页点击登出按钮注销
- 若忘记密码，则点击忘记密码按钮，填写注册的邮箱，系统会发邮件给用户告知之前所设密码
